# NB Visual

Generate a directed graph of nb doc links and render it in the browser.

## Usage

```bash
python nb-visual/build_nb_graph.py
```

Open `nb-visual/index.html` in a browser. Re-run the build script after nb docs change to refresh `graph.json`.
